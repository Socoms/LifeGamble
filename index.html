🎲 다이스 게임 학습 가이드

게임 규칙
  • 숫자 선택: 1부터 99까지의 숫자를 슬라이더로 선택합니다
  • 배팅 옵션: 선택한 숫자보다 "더 작은" 또는 "더 큰" 숫자가 나올지 예측합니다
  • 게임 진행: 0부터 100까지의 랜덤 숫자가 생성됩니다
  • 승리 조건: 예측한 대로 숫자가 나오면 승리, 그렇지 않으면 패배입니다
확률 계산
  • 더 작은 확률: 선택한 숫자 / 100 × 100%
  • 더 큰 확률: (100 - 선택한 숫자) / 100 × 100%
  • 예: 숫자 50을 선택하면 더 작은 확률 50%, 더 큰 확률 50%
  • 예: 숫자 30을 선택하면 더 작은 확률 30%, 더 큰 확률 70%
배당률 계산
  • 배당률 = 1 / 확률 × (1 - 하우스 엣지)
  • 하우스 엣지: 1% (카지노 우위)
  • 확률이 낮을수록 배당률이 높아집니다
  • 예: 50% 확률 → 약 x1.98 배당
  • 예: 30% 확률 → 약 x3.30 배당
학습 목표
  • 확률과 배당률의 관계 이해
  • 기대값 계산 방법 학습
  • 하우스 엣지(카지노 우위) 이해
  • 리스크 관리 개념 학습

🃏 하이로우 카드 게임 학습 가이드

게임 규칙
  • 현재 카드를 보고 다음 카드가 높을지 낮을지 배팅합니다
  • 높음: 다음 카드가 현재 카드보다 높으면 승리
  • 낮음: 다음 카드가 현재 카드보다 낮으면 승리
  • 동일한 카드가 나오면 보통 무효 처리되거나 특별 규칙 적용
  • 카드 교체 기능으로 새로운 카드로 시작할 수 있습니다
카드 값
  • A(에이스) = 1 (가장 낮음)
  • 2-10 = 숫자 그대로
  • J(잭) = 11, Q(퀸) = 12, K(킹) = 13 (가장 높음)
확률 계산
  • 남은 카드 수에 따라 확률이 실시간으로 변합니다
  • 현재 카드가 높을수록 낮음 배팅의 확률이 높아집니다
  • 현재 카드가 낮을수록 높음 배팅의 확률이 높아집니다
  • 조건부 확률의 개념을 학습할 수 있습니다
학습 목표
  • 조건부 확률 이해
  • 카드 확률 계산
  • 전략적 의사결정
  • 기대값 계산

🎰 룰렛 게임 규칙

게임 개요
  • 유럽식 룰렛은 0부터 36까지 총 37개의 숫자가 있습니다
  • 0은 녹색, 나머지 숫자는 빨강 또는 검정으로 구분됩니다
  • 공을 돌리고 어느 숫자에 떨어지는지 예측하여 배팅합니다
배팅 옵션
  • 직접 배팅: 특정 숫자에 배팅 (배당 35:1)
  • 분할 배팅: 인접한 2개 숫자 (배당 17:1)
  • 거리 배팅: 3개 숫자 (배당 11:1)
  • 코너 배팅: 4개 숫자 (배당 8:1)
  • 라인 배팅: 6개 숫자 (배당 5:1)
  • 더즌: 12개 숫자 (배당 2:1)
  • 컬럼: 12개 숫자 (배당 2:1)
  • 1-18 / 19-36: 하이/로우 (배당 1:1)
  • 홀수 / 짝수: (배당 1:1)
  • 빨강 / 검정: (배당 1:1)
확률 및 기대값
  • 직접 배팅 승률: 1/37 (2.70%), 기대값: -2.70%
  • 홀/짝, 빨강/검정 승률: 18/37 (48.65%), 기대값: -2.70%
  • 더즌/컬럼 승률: 12/37 (32.43%), 기대값: -2.70%
  • 모든 배팅의 하우스 엣지는 동일하게 2.70%입니다

🎴 텍사스 홀덤 게임 규칙

게임 개요
  • 최대 6명의 플레이어가 함께 플레이하는 멀티플레이어 포커 게임입니다
  • 각 플레이어는 2장의 개인 카드(홀 카드)를 받습니다
  • 딜러가 5장의 커뮤니티 카드를 공개합니다
  • 7장의 카드 중 5장을 조합하여 최고의 패를 만듭니다
게임 진행
  • 프리플롭: 홀 카드 2장을 받고 첫 번째 베팅 라운드
  • 플롭: 커뮤니티 카드 3장 공개, 두 번째 베팅 라운드
  • 턴: 커뮤니티 카드 4장째 공개, 세 번째 베팅 라운드
  • 리버: 커뮤니티 카드 5장째 공개, 마지막 베팅 라운드
  • 쇼다운: 남은 플레이어들이 패를 공개하고 승자 결정
베팅 액션
  • 체크: 베팅하지 않고 차례를 넘김 (베팅이 없을 때만 가능)
  • 콜: 현재 베팅 금액에 맞춤
  • 레이즈: 베팅 금액을 올림
  • 폴드: 패를 버리고 게임에서 나감
패 순위 (높은 순)
  • 로얄 플러시 (A-K-Q-J-10, 모두 같은 무늬)
  • 스트레이트 플러시 (연속된 5장, 모두 같은 무늬)
  • 포카드 (같은 숫자 4장)
  • 풀하우스 (트리플 + 페어)
  • 플러시 (같은 무늬 5장)
  • 스트레이트 (연속된 5장)
  • 트리플 (같은 숫자 3장)
  • 투 페어 (페어 2개)
  • 원 페어 (페어 1개)
  • 하이 카드 (아무것도 없을 때 가장 높은 카드)
