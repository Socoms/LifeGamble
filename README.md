# 📊 확률과 전략 학습 게임

확률, 통계, 수학적 전략을 배우는 교육용 시뮬레이터입니다.

## 🎯 프로젝트 소개

바카라와 블랙잭의 수학적 원리를 이해하고 최적의 전략을 학습할 수 있는 교육용 게임입니다. 실제 도박이 아닌 학습 목적으로 제작되었습니다.

## ✨ 주요 기능

### 📈 바카라 확률 학습
- 이론적 승률과 기대값 계산
- 실시간 확률 분석
- 통계 기록 및 분석
- 하우스 엣지(카지노 우위) 이해

### 🧮 블랙잭 전략 학습
- 기본 전략 가이드
- 확률 기반 의사결정 연습
- 최적 전략 시뮬레이션
- 기대값 계산 학습

## 🎮 게임 모드

1. **바카라 확률 학습**
   - 플레이어 vs 뱅커 승률 비교
   - 타이 확률 분석
   - 실시간 통계 표시

2. **블랙잭 전략 학습**
   - 히트/스탠드 전략
   - 더블다운 전략
   - 최적 의사결정 연습

## 📚 학습 목표

- 확률과 통계의 이해
- 수학적 전략의 활용
- 기대값 계산 방법
- 리스크 관리 개념
- 금융 리터러시 향상

## 🛠️ 기술 스택

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Deck of Cards API

## 📖 사용 방법

1. `index.html` 파일을 웹 브라우저에서 열기
2. 학습 모드 선택 (바카라 또는 블랙잭)
3. 학습 포인트를 선택하고 전략 테스트
4. 확률 분석 및 통계 확인

## ⚠️ 주의사항

이 게임은 **교육 목적**으로만 제작되었습니다. 실제 도박을 권장하지 않으며, 모든 게임은 마이너스 기대값을 가지고 있음을 학습 목적으로 보여줍니다.

## 📝 라이선스
